## 🚀 Features

### 👥 User Management
- Secure Registration & Login (Token-based Auth)
- Role-based access: Students & Alumni
- Profile creation with education, work experience, and avatar

### 🔍 Find Alumni
- Search and filter alumni by:
  - Name
  - Company
  - Year of graduation
  - Location
- View detailed profiles and initiate conversation

### 💬 Chat System
- One-on-one private messaging
- Real-time chat using Socket.io
- Typing indicators
- Read receipts ✓✓
- Last seen status
- Grouping messages by date
- Global public chat

### 📄 Job Board
- Alumni can post job opportunities
- Students can:
  - View jobs
  - Upload resumes (PDF/DOC/DOCX/Images)
  - Apply to jobs (with 'Applied' status)
- Job posters can view applicants and their resumes
- Option to edit/delete job posts

### 💸 Donate & Sponsor
- Razorpay integration for secure payments
- Distinct flows for donations and sponsorships

### 🤖 AI Chatbot
- Smart assistant powered by OpenAI/Gemini
- Assists users with platform navigation and queries
